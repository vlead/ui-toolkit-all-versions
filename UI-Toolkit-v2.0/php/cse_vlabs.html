Computer Science Virtual Labs
This page leads you to the computer science labs and other labs built at IIIT-H. Virtual Labs is an initiative of the Ministry of Human Resource Development (MHRD), Govt. of India , under the National Mission on Education through ICT. For more details, please visit Sakshat virtual labs.
  • Exploring the Unexplored : - To provide remote-access to Labs in Computer Science and Engineering. These Virtual Labs would cater to students at the undergraduate level, post graduate level as well as to research scholars.
  • Scientific Temperament:- To enthuse students to conduct experiments by the arousing their curiosity. This would help them in learning basic and advanced concepts through remote experimentation.
  • Sustainable Learning Skills:- To provide a complete Learning Management System around the Virtual Labs where the students can avail the various tools for learning, including additional web-resources, video-lectures, animated demonstrations and self evaluation.
  • Minimize the cost of building labs:- To share costly equipment and resources, which are otherwise available to limited number of users due to constraints on time and geographical distances.

Pre-requisites System Requirements to run Virtual Labs
Java
Java allows you to play online virtual labs.
Java 3D
The Java 3D API enables the creation of three-dimensional graphics applications and Internet-based 3D applets. It provides high-level constructs for creating and manipulation 3D geometry and building the structures used in rendering that geometry. With this software, you can efficiently define and render very large virtual worlds.
Flash Player
Adobe Flash (formerly Macromedia Flash) is a multimedia platform used to add animation, video, and interactivity to web pages.

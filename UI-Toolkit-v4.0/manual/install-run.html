Installing and Running

Download the Graphics VirtualLab package from the Experiment tab.
Or click here to download

On Windows or Mac OSX:

  1. Unzip the downloaded zip file.
  2. Open the
  3. VirtualLabGraphics
    4. folder you just extracted.
  4. Launch the application by double clicking the Start.jar file in the VirtualLabGraphics folder.
  5. Choose the experiment to start by clicking it.

On Linux:

  1. Extract the downloaded zip file to VirtualLabGraphics folder.
    Or Unzip the zip file from terminal.
     $ unzip VirtualLabGraphics.zip
  2. Go to the VirtualLabGraphics directory from terminal.
     $ cd VirtualLabGraphics/
  3. Launch the application by executing Start.jar using the command below in the VirtualLabGraphics directory.
     $ java -jar Start.jar
  4. Choose the experiment to start by clicking it.

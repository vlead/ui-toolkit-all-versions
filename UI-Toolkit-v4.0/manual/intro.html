Introduction

Graphics Virtual Lab tool provides a platform to learn various concepts of Computer Graphics. This comes preloaded with six different experiments. Start the respective application to understand the concepts practically. Each experiment comes preloaded with required geometry and the corresponding theory and guidance to understand the concepts better.

The list of experiments packaged with the application:

  1. Points and Coordinate Systems
  2. Transformations: Translation
  3. Transformations: Rotation
  4. Transformation: Scaling
  5. Hierarchical Transformations
    1. 2D Demo
    2. 3D Articulated Arm
  6. Projections and Cameras

Please refer the website for more details on each experiment. You can however use the entire features of the application and try your own experiments by just starting with any provided experiment.

Hierarchical Structure Display Panel

This section displays the different cameras and instances created in the scene and the entire hierarchical structure under the world title. Please refer the corresponding topics for detailed explanation on each element.

The buttons in the Hierarchical structure display section provide ways to add New instances, Edit any selected feature, Delete a node and Add a camera to the scene. Each of these buttons open a new window where the parameters for the corresponding node are entered.
